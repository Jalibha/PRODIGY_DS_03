## Internship - Prodigy InfoTech (Data Science)

- Scikit-learn

### Learning Outcomes:
- Preprocessing text data
- Applying NLP techniques
- Sentiment classification

### Output:
A CSV file with tweets and their corresponding sentiment labels.
